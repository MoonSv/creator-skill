# Creator Skill - 小红书内容创作助手

基于小红书实时数据，帮你生成完整的发布建议（标题、图片选择、封面、内容优化）。

## 安装（一条命令）

```bash
git clone https://github.com/MoonSv/creator-skill.git && cd creator-skill && ./install.sh
```

安装脚本会自动完成：
- 安装 xiaohongshu-cli（小红书数据采集）
- 安装 Pillow（封面图生成）
- 配置 PATH
- 注册 Claude Code Plugin
- 引导扫码登录小红书（建议用小号）

## 使用

安装后，在 Claude Code 中直接说：

```
我想发一条关于「北京探店咖啡馆」的小红书，图片在 ~/Photos/cafe/
```

自动完成所有分析，最终在浏览器中打开一份交互式 HTML 报告。

## 输出内容

| 板块 | 内容 |
|------|------|
| 爆款分析 | Top帖子数据对比、爆款标题模式 |
| 大V分析 | 该领域博主排行、可借鉴策略 |
| 图片推荐 | 封面选择、发布序列、排除理由 |
| 封面生成 | 4种风格自动匹配生成 |
| 内容建议 | 标题候选、正文模板、话题标签 |
| 策略洞察 | 爆火因素、差异化角度、风险提醒 |
