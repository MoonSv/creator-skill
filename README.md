# Creator Skill - 小红书内容创作助手

基于小红书实时数据的 Claude Code Plugin，三大功能助你做好小红书。

## 安装（一条命令）

```bash
git clone https://github.com/MoonSv/creator-skill.git && cd creator-skill && ./install.sh
```

安装脚本会自动完成：
- 安装 xiaohongshu-cli（小红书数据采集）
- 安装 Pillow（封面图生成）
- 配置 PATH
- 注册 Claude Code Plugin
- 引导扫码登录小红书（建议用小号）

## 三大功能

### 1. 发布建议

```
我想发一条关于「北京探店咖啡馆」的小红书，图片在 ~/Photos/cafe/
```

| 板块 | 内容 |
|------|------|
| 爆款分析 | Top帖子数据对比、爆款标题模式 |
| 大V分析 | 该领域博主排行、可借鉴策略 |
| 图片推荐 | 封面选择、发布序列、排除理由 |
| 封面生成 | 4种风格自动匹配生成 |
| 内容建议 | 标题候选、正文模板、话题标签 |
| 策略洞察 | 爆火因素、差异化角度、风险提醒 |

### 2. 帖子诊断

```
帮我分析下这条笔记为什么数据不好 https://www.xiaohongshu.com/explore/xxx
```

| 板块 | 内容 |
|------|------|
| 数据健康度 | 互动数据评分、收藏/点赞比、同主题百分位 |
| 竞品对比 | vs Top 3 竞品逐项对比，找出差距 |
| 五维诊断 | 标题/内容/封面/标签/时机逐一分析 |
| 行动建议 | 立即可改 + 下次改进 + 数据预期 |

### 3. 账号分析

```
分析我的小红书账号
```

| 板块 | 内容 |
|------|------|
| 账号画像 | 量级定位、互动率、发布频率、趋势 |
| 爆款复盘 | Top/Bottom 帖子对比、爆款共性提炼 |
| 对标博主 | 同领域博主数据对比、可借鉴策略 |
| 运营建议 | 内容方向、发布节奏、增长策略、行动清单 |

## 安全设计

只读操作，不发帖/点赞/评论。限速 3-5 秒/请求。建议小号采集。
